# rpi

A command line tool for extracting and analyzing Soccer data

## Installation

```bash
pip install -e .
```


## Usage

Install the pre-commit hooks

```bash
pre-commit install
```

Run the hooks manually on all files

```bash
pre-commit run --all-files
```
